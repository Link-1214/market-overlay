Market Overlay v2.1 패치노트

이번 문서는 v2.0 이후 실제로 변경된 내용만 정리했습니다.

1. 시장 데이터 안정성

- Yahoo 일봉에서 직전 거래일 종가가 일시적으로 비어 있으면 같은 chart API의 60분봉으로 전일가를 조건부 보정합니다.
- 정상 일봉 응답에서는 추가 조회하지 않아 요청량을 늘리지 않습니다.
- CNN Fear & Greed는 브라우저 유사 헤더로 조회하고, 실패 시 기존 Local F&G 대체 계산을 유지합니다.

2. 일본 종목과 JPY 지원

- 일본 4자리 종목코드와 Yahoo `.T` 심볼을 지원합니다.
- 자동완성은 한국, 미국, 일본 순으로 정렬합니다.
- 포트폴리오의 JPY 평단가와 예수금을 JPY/KRW 환율로 원화 환산합니다.
- 매매/배당 요약 카드는 늘리지 않고 JPY 기록을 총액에 원화 환산해 반영합니다.

3. 트리맵과 성과비교 안내

- 기존 `히트맵` 명칭을 실제 시각화 방식에 맞게 `트리맵`으로 통일했습니다.
- 미국 종목은 티커, 한국/일본 종목은 종목명을 박스에 표시합니다.
- 박스 크기별 라벨 생략과 tooltip을 적용해 작은 박스의 글자 겹침을 줄였습니다.
- 성과비교와 트리맵에 반영 항목 수, 제외 항목 수와 주요 제외 사유를 표시합니다.

4. 업데이트 확인

- 실행 후 하루 한 번 public GitHub의 최신 정식 Release를 확인합니다.
- 업데이트가 없거나 자동 조회가 실패하면 팝업을 표시하지 않습니다.
- 새 버전이 있으면 릴리즈 페이지 열기, 이번 버전 알림 끄기, 나중에를 선택할 수 있습니다.
- 설정 탭에서 자동 확인을 끄거나 수동 확인하고, 숨긴 버전 알림을 다시 켤 수 있습니다.
- 자동 다운로드와 자동 설치는 하지 않습니다.

5. 백업/복원 안내

- 백업 대상, 제외 대상, 현재 데이터 교체 방식과 자동 사전 백업을 더 명확히 안내합니다.
- 업데이트 확인 설정도 백업/복원 대상에 포함됩니다.

6. 내부 구조와 종료 안정성

- 시각화 정책/위젯, 시장·종합·내자산 오버레이, 데이터 worker, 단일 인스턴스 생명주기를 책임별 모듈로 분리했습니다.
- 앱은 이전처럼 하나의 실행 파일로 배포됩니다.
- 종료 시 타이머, worker, 로컬 서버와 mutex를 정리해 오버레이만 남는 문제를 방어했습니다.
- v2.0 state의 포트폴리오, 매매/배당, JPY, 사용자 표시명과 트리맵 설정 이관을 회귀 테스트했습니다.

주의

- 이 앱은 투자 판단을 대신하지 않는 보조 도구입니다.
- Yahoo Finance/CNN JSON 데이터는 증권사 HTS/MTS와 지연, 환율, 세금, 수수료 기준이 다를 수 있습니다.
- 코드 서명이 없는 빌드는 Windows SmartScreen 또는 보안 프로그램 경고가 표시될 수 있습니다.
