# Market Overlay v2.0

v2.0은 v1.9 이후 유지보수성과 배포 안정성을 정리한 릴리즈입니다.

## 주요 변경

- 코드 구조 분리
  - 계산, 상태 저장, 포맷, 종목명, 히스토리 캐시, worker payload, UI 조립 책임을 분리했습니다.
- 설치/제거 UX 보강
  - 기존 설치 감지와 제거 흐름을 보강했습니다.
  - 제거 시 개인 데이터 범위를 안내합니다.
- 백업/복원 안내 개선
  - 백업 내보내기/가져오기 문구를 정리했습니다.
  - import 실패 시 기존 데이터 복구 흐름을 검증했습니다.
- 종목명 데이터 갱신
  - 내장 종목명 데이터를 갱신했습니다.
  - `SPCX`는 `스페이스X`로 매핑했습니다.
- 매매/배당 기록 UX 보강
  - 필터 초기화 버튼을 추가했습니다.
  - 배당 불러오기 0건 안내를 개선했습니다.
- 문서/배포 운영 정리
  - public repo는 설치파일/사용자 문서 중심으로 유지합니다.
  - private repo는 소스와 개발 문서를 관리합니다.

## 주의

- 이 앱은 투자 판단을 대신하지 않는 보조 도구입니다.
- Yahoo Finance/CNN JSON 기반 조회값은 증권사 HTS/MTS와 지연, 환율, 세금, 수수료 기준이 다를 수 있습니다.
- 코드 서명이 없는 빌드라 Windows SmartScreen 또는 보안 프로그램 경고가 표시될 수 있습니다.
