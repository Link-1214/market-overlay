Market Overlay v1.9

이번 버전은 v1.8에서 이어지는 안정화 + 내 포트폴리오/종합 오버레이 추가 버전입니다.

다운로드 링크

업로드 후 링크 삽입

패치 내용

1. 오버레이 모드 추가

- 설정 탭에서 `시장 오버레이` / `종합 오버레이` / `내 포트폴리오 오버레이` 선택 가능
- 종합 오버레이는 시장 상태와 내 자산 요약을 한 화면에서 같이 보는 압축 모드입니다.
- 시장요약, 환율, 공탐, VIX, S&P, 나스닥, 코스피, 총평가액, 원금, 수익, 전일대비가 같이 나옵니다.

2. 내 포트폴리오 오버레이 추가

- 기존 시장 지표 오버레이 말고 내 포트폴리오 상태를 오버레이로 볼 수 있게 했습니다.
- 설정 탭에서 `내 포트폴리오 오버레이`를 선택하면 됩니다.
- 내 포트폴리오 오버레이에는 손익률 바, 포트 비중 도넛, 투자/현금 비중 도넛, 총평가액/원금/수익/전일대비 카드가 나옵니다.
- 포트폴리오 탭에서 계산하는 종합 수치와 같은 기준을 씁니다.

3. 중복 실행 방지

- 이미 실행 중인데 바로가기를 다시 눌러도 앱이 여러 개 뜨지 않게 했습니다.
- 새 실행은 기존 컨트롤 창을 앞으로 가져오는 방식으로 동작합니다.
- 오버레이만 남거나 컨트롤러가 꼬이는 상황을 줄이기 위한 안정화입니다.

4. 데이터 경로 정리

- 사용자 개인 데이터는 계속 `%APPDATA%\MarketOverlay`에 보존됩니다.
- 업데이트 설치로 포트폴리오, 매매기록, 배당기록, 사용자 표시명이 날아가지 않도록 설치 전후 검증했습니다.

주의

- 코드 서명이 없어서 SmartScreen이나 보안 프로그램 경고가 뜰 수 있습니다.
- 이 프로그램은 투자 판단을 대신하는 도구가 아니라 포트폴리오와 시장 흐름을 빠르게 확인하는 보조 도구입니다.
- Yahoo Finance 기반 데이터라 증권사 화면과 약간의 지연, 환율, 세금, 수수료 차이가 있을 수 있습니다.
