Market Overlay v1.8 배포함

지난번 1.7에서 성과비교랑 포트폴리오 쪽을 붙였는데, 막상 실사용해보니까 장부 입력이랑 시각화 쪽이 꽤 불편했음.

그래서 이번 1.8은 기능 추가도 있긴 한데, 핵심은 "내가 실제로 쓰면서 덜 빡치게 만드는 버전" 쪽임.

다운로드 링크

여기에 링크 넣으면 됨


대충 뭐하는 프로그램임?

시장 상태랑 내 포트폴리오를 데스크톱에서 가볍게 보는 보조 도구임.

시장 쪽은 공포탐욕지수, VIX, 환율, 나스닥/슨피/코스피 같은 지수 흐름을 보고,
포트폴리오 쪽은 내가 들고 있는 종목, 평가액, 수익, 전일대비, 현금비중 같은 걸 같이 봄.

초단타용 프로그램은 아니고 5분~10분 단위로 시장이랑 내 자산 상태 체크하는 용도라고 보면 됨.


v1.8에서 바뀐 것

1. 포트폴리오 화면 정리

- 카드 아래 화면을 좌우 2분할로 바꿈
- 왼쪽은 시각화, 오른쪽은 보유 목록
- 시각화는 비중 도넛 / 히트맵 토글로 볼 수 있음
- 도넛은 내 자산 비중을 대략적으로 보기 좋게 표시함
- 히트맵은 종목별 비중과 기간 수익률을 한 번에 보여줌

히트맵은 박스 크기가 현재 보유 비중이고,
색은 선택한 기간의 수익률임.

기간은 전일, 1주, 1개월, 3개월, 6개월, YTD, 1년 지원함.

마우스 올리면 종목명, 종목코드, 수익률, 비중, 평가액도 툴팁으로 뜸.


2. 매매기록 쪽 개선

이제 매매기록을 그냥 실현수익 적는 곳이 아니라,
나중에 배당이나 성과 계산에 쓸 수 있는 장부처럼 정리함.

- 매수/매도 기록 관리
- 포트, 구분, 통화는 드롭다운
- 종목은 자동완성 입력
- 종목명, 1주당 손익, 총손익, 수익률은 자동 계산
- 필터랑 정렬 지원
- 보이는 항목 전체 선택/삭제 가능

그리고 예수금이 부족해도 매수가 막히지 않게 바꿈.
실제로 여러 거래 입력할 때 예수금부터 맞춰놓고 입력하는 게 꽤 귀찮아서,
음수 예수금도 허용하고 대신 현금비중 카드에서 빨간색으로 경고 뜨게 했음.


3. 성과비교 개선

성과비교 탭도 손봤음.

현재 보유수량과 현재 포트폴리오 구성을 선택한 기간 동안 그대로 들고 있었다고 가정해서,
내 포트폴리오랑 IXIC, SPX, KOSPI, 금, 비트코인 흐름을 비교할 수 있음.

기간은 1개월, 3개월, 6개월, 1년, 3년.

이건 정확한 입출금 반영 백테스트가 아니라,
"지금 내 구성으로 보면 최근 구간에서 대충 어땠나" 보는 단순 비교임.

데이터가 부족한 종목은 제외하고 제외 사유도 표시함.


4. 반응성/캐시 개선

성과비교랑 히트맵은 매번 데이터를 새로 부르는 게 아니라,
10분 정도는 캐시를 재사용하도록 바꿈.

버튼 이름은 새로고침이지만, 무조건 인터넷 다시 조회하는 게 아니라
가능하면 이미 가져온 데이터를 기준으로 다시 계산함.

그래서 이전보다 덜 답답하게 쓰는 쪽으로 조정했음.


5. 기타

- 컨트롤러가 안 보이고 오버레이만 남는 문제 완화
- 배당/매매기록 화면에서 불필요한 메모 칼럼 제거
- 드롭다운 위에서 마우스 휠 굴렸다고 값이 바뀌는 문제 방지
- 설치 전후 개인 데이터 보존 확인
- 배포 전 자동 체크 스크립트 추가


주의사항

이 프로그램은 투자 판단을 대신해주는 프로그램이 아님.

시장 데이터는 Yahoo Finance랑 CNN F&G 쪽 데이터를 기반으로 가져오고,
증권사 HTS/MTS의 확정 수치랑 다를 수 있음.

세금, 수수료, 환율, 체결 오차 같은 건 증권사 기준이 최종임.

그리고 아직 개인이 만든 테스트성 툴이라 환경에 따라 버그가 있을 수 있음.
피드백 주면 다음 버전에 반영해보겠음.
