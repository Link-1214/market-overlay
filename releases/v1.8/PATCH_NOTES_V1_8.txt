Market Overlay v1.8 패치노트

이번 버전은 v1.7 이후 오너 실사용 중 불편했던 매매기록, 성과비교, 포트폴리오 시각화, 반응성 문제를 중심으로 정리한 빌드입니다.

1. 포트폴리오 입력 개선
- 매수 시 예수금이 부족해도 거래 입력을 막지 않도록 변경했습니다.
- 여러 거래를 실제 입력 순서대로 넣기 쉽도록 예수금 음수를 허용합니다.
- 예수금이 음수일 때 현금비중 카드는 빨간색 경고로 표시됩니다.
- 포트폴리오 화면은 카드 아래를 좌측 시각화, 우측 목록의 2분할 구조로 정리했습니다.

2. 매매기록 원장화
- 매매기록 탭은 기간별 보유수량, 배당 권리, 성과 계산의 기준 원장으로 쓰는 방향으로 정리했습니다.
- 포트는 포트폴리오 ID와 연결되는 드롭다운으로 선택합니다.
- 구분은 매수/매도만 허용합니다.
- 종목 입력은 포트폴리오 탭과 같은 자동완성 방식으로 바꿨습니다.
- 일자, 종목, 매수단가, 매도단가, 수량은 직접 입력할 수 있고, 종목명/1주당 손익/총손익/수익률은 자동 계산 또는 매핑 결과로만 표시합니다.
- 매매기록과 배당 탭의 메모 칼럼은 화면 복잡도를 줄이기 위해 숨겼습니다.

3. 매매기록/배당 필터 개선
- 포트, 구분, 통화, 상태, 종목명, 기간 필터를 추가 또는 정리했습니다.
- 상단 손익 카드는 현재 필터 결과 기준으로 합계를 표시합니다.
- 보이는 항목 전체 선택과 선택 삭제를 지원합니다.
- 매매기록/배당 탭의 드롭다운은 목록이 열린 상태가 아니면 마우스 휠로 값이 바뀌지 않게 했습니다.

4. 성과비교 개선
- 성과비교는 현재 보유수량과 현재 포트폴리오 구성을 선택 기간 내내 유지했다고 가정한 단순 비교로 정리했습니다.
- 기간은 1개월, 3개월, 6개월, 1년, 3년을 지원합니다.
- IXIC, SPX, KOSPI, 금, 비트코인을 체크박스로 비교할 수 있습니다.
- 그래프에 0% 기준선을 표시합니다.
- 데이터가 부족하거나 조회 실패한 종목은 제외하고, 제외 사유를 화면에 표시합니다.
- 성과비교와 히트맵 데이터는 10분 TTL 메모리 캐시를 재사용해 불필요한 재조회와 대기 시간을 줄였습니다.

5. 포트폴리오 시각화
- 포트폴리오 탭에 비중 도넛과 기간별 히트맵을 추가했습니다.
- 히트맵은 종목 박스 크기=현재 평가액 비중, 색상=선택 기간 수익률입니다.
- 기간은 전일, 1주, 1개월, 3개월, 6개월, YTD, 1년을 지원합니다.
- 히트맵 색상 기준은 기간별로 다르게 적용합니다.
- 히트맵 hover tooltip은 종목명, 종목코드, 선택 기간 수익률, 비중, 평가액을 표시합니다.
- 종합 히트맵에서 종목을 눌러 포트 상세로 이동한 뒤, 시각화 영역의 종합 복귀 버튼으로 돌아갈 수 있습니다.

6. 안정성/검증
- 컨트롤러가 숨겨지고 오버레이만 남는 현상을 줄이기 위해 시작 시 컨트롤 창 복구 로직을 보강했습니다.
- 배포 전 자동 체크 스크립트를 추가했습니다.
- 계산 정책 문서와 구조 분리/최적화 메모를 추가했습니다.
- 최신 빌드는 설치 전후 사용자 state JSON 해시 보존을 확인했습니다.

주의
- v1.8은 기능이 크게 늘어난 빌드입니다. 성과비교와 히트맵은 투자 판단을 대신하는 도구가 아니라 보유 상태를 빠르게 보기 위한 보조 기능입니다.
- 시장 데이터는 Yahoo Finance chart API와 CNN F&G dataviz JSON을 기반으로 하며, 증권사 HTS/MTS 확정값과 다를 수 있습니다.
- 세금, 수수료, 체결 오차, 정확한 실현손익 확정 계산은 증권사 내역과 차이가 있을 수 있습니다.
