Market Overlay v2.2 사용자 가이드
=================================

1. 설치와 업데이트

- 기존 버전이 설치되어 있으면 v2.2 설치본을 실행해 같은 경로에 업데이트 설치합니다.
- 포트폴리오, 매매기록, 배당기록, 사용자 표시명, 설정은 `%APPDATA%\MarketOverlay`에 보관되므로 업데이트 설치 후에도 유지됩니다.
- 제거할 때만 개인 데이터를 삭제할지 선택합니다.

2. 업데이트 확인

- 설정 탭에서 현재 버전과 업데이트 확인 상태를 볼 수 있습니다.
- 자동 업데이트 확인은 기본 켜짐이며, 앱 실행 후 하루에 한 번 public GitHub Release를 확인합니다.
- 새 버전이 있으면 GitHub Release 페이지로 이동할 수 있습니다.
- 앱이 자동으로 다운로드하거나 설치하지 않습니다.

3. 시장 데이터

- 시장 탭은 Yahoo Finance chart API와 CNN Fear & Greed JSON을 사용합니다.
- Yahoo 일봉에서 전일 종가가 잠시 비면 5일 60분봉 데이터로 보정합니다.
- CNN F&G가 실패하면 Local F&G 대체 계산값을 표시합니다.
- 데이터 조회가 실패해도 이전 정상값이 있으면 마지막 정상값을 유지합니다.

4. 일본 종목과 JPY

- 일본 상장 종목은 Yahoo Finance 형식인 `.T`를 사용합니다. 예: `7203.T`
- 숫자만 입력해도 내장 데이터에 있으면 일본 종목 후보를 자동완성합니다.
- 일본 종목 평단가와 예수금 통화로 JPY를 사용할 수 있습니다.
- 매매기록과 배당 요약 카드는 복잡도를 줄이기 위해 별도 JPY 카드를 만들지 않고 원화 환산 합계에 포함합니다.

5. 포트폴리오 비중/트리맵

- 포트폴리오 탭 카드 아래는 왼쪽 시각화 영역과 오른쪽 목록 영역으로 나뉩니다.
- 가운데 분할선을 드래그하면 왼쪽 비중/트리맵 영역과 오른쪽 목록 영역의 폭을 조절할 수 있습니다.
- `비중`은 현재 평가액 기준 비중 도넛 차트입니다.
- `트리맵`은 박스 크기로 현재 자산 비중을, 색상으로 선택 기간 수익률을 보여줍니다.
- `기본값` 버튼을 누르면 시각화/목록 폭이 기본 비율로 돌아갑니다.

6. 진단 로그

- 설정 탭의 `로그 폴더 열기` 버튼으로 진단 로그 폴더를 열 수 있습니다.
- 로그는 네트워크 실패, 파싱 실패, state 실패, worker 실패를 추적하기 위한 파일입니다.
- 보유수량, 평단가, 예수금 같은 장부 세부값은 로그에 남기지 않습니다.

7. 백업과 복원

- 설정 탭에서 사용자 데이터를 내보내거나 가져올 수 있습니다.
- 백업에는 포트폴리오, 매매기록, 배당기록, 사용자 표시명, 설정, schema 정보가 포함됩니다.
- 가져오기 실패 시 기존 데이터가 부분적으로 덮이지 않도록 기존 state를 보존합니다.

8. 주의

- Market Overlay는 투자 판단을 대신하지 않는 보조 도구입니다.
- 실시간 초단타용 도구가 아니라 5~10분 단위 시장과 포트폴리오 흐름 확인용입니다.
- 증권사 HTS/MTS, 세금, 수수료, 환율 기준과 차이가 날 수 있습니다.
