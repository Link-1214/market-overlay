Market Overlay v2.2 올림
========================

이번 버전은 기능을 많이 늘린 버전이라기보다는, 포트폴리오 시각화 화면을 더 쓰기 편하게 만들고 내부 신뢰성/진단을 보강한 릴리즈에 가까움.

사용자가 바로 체감할 제일 큰 변경은 포트폴리오 탭의 비중/트리맵 영역임.

1. 포트폴리오 비중/트리맵 화면 개선
- 포트폴리오 탭 카드 아래 영역을 왼쪽 시각화, 오른쪽 보유 목록으로 나눴음.
- 가운데 분할선을 드래그해서 비중/트리맵 영역을 넓히거나 줄일 수 있음.
- 기본값 버튼으로 원래 보기 좋은 비율로 되돌릴 수 있고, 조절한 폭은 다음 실행 때도 유지됨.
- 트리맵은 박스 크기 = 현재 자산 비중, 색상 = 선택 기간 수익률.
- 텍스트가 겹치거나 잘리는 경우를 줄이고, 작은 박스는 tooltip으로 상세 정보를 확인할 수 있게 정리함.

2. 데이터 조회 구조 정리
- Yahoo 원본 일봉에서 전일 종가가 잠시 비면 전일대비가 이상하게 나올 수 있어서, 이 보정은 v2.1에서 이미 넣어둔 상태였음.
- v2.2에서는 그 보정 로직을 새로 만든 게 아니라, Yahoo/CNN/환율 조회를 별도 provider로 빼고 기존 보정 경로가 계속 유지되는지 테스트와 로그로 묶었음.
- 이후 시장 데이터 오류가 생겼을 때 어느 조회 경로에서 문제가 났는지 추적하기 쉬워짐.

3. 진단 로그 추가
- 문제가 생겼을 때 원인을 좀 더 찾기 쉽게 로그를 남기도록 함.
- 단, 보유수량/평단가/예수금 같은 장부 세부값은 로그에 남기지 않도록 처리함.
- 설정 탭에서 로그 폴더를 열 수 있음.

4. 종목명 데이터 최신화
- 미국 상장, KRX 상장법인/ETF/ETN, 일본 JPX 종목 데이터를 다시 수집함.
- 총 21,558개 종목명 매핑.

5. 내부 구조 정리
- 시각화, 오버레이 3종, 데이터 worker, 시장 데이터 조회, state 이관, 진단 로그, 중복실행/종료 처리를 각각 더 분리함.
- 앱은 여전히 하나의 Windows 설치 프로그램으로 설치됨.

주의
- 투자 판단을 대신하는 도구는 아님.
- Yahoo Finance 기반이라 증권사 HTS/MTS와 지연, 환율, 세금, 수수료 기준 차이가 있을 수 있음.
- 코드서명 없는 개인 빌드라 SmartScreen이나 보안 프로그램 경고가 뜰 수 있음.

기존 사용자 데이터는 업데이트 설치해도 유지됨.
