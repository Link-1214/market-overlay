Market Overlay v2.0 올림

이번 버전은 막 화려하게 기능이 확 늘어난 버전이라기보다는,
그동안 계속 쌓이던 코드랑 설치/배포 쪽을 한 번 정리한 버전임.

다운로드는 GitHub Release에서 받으면 됨.

주요 변경

1. 설치/제거 쪽 정리

- 기존 설치가 있으면 설치/업데이트, 제거, 종료를 더 명확하게 고를 수 있게 함
- 제거할 때 개인 데이터가 뭔지 안내하고 보존/삭제 판단할 수 있게 정리
- 기존 설치 감지를 좀 더 튼튼하게 보강

2. 코드 구조 정리

- 겉보기엔 크게 티 안 나는데 내부 구조를 많이 나눴음
- 계산, 저장, 종목명, 히스토리 캐시, UI 조립 같은 것들을 분리해서 다음 업데이트 때 덜 꼬이게 만듦
- 이게 사실 이번 버전의 제일 큰 작업임

3. 백업/복원 안내 개선

- 백업 내보내기/가져오기 안내를 좀 더 명확하게 바꿈
- 가져오기 실패 시 기존 데이터 복구 흐름도 체크함

4. 종목명 데이터 갱신

- 종목명 사전 갱신함
- SPCX는 스페이스X로 따로 매핑해둠
- 사용자가 직접 고친 표시명은 계속 우선 적용됨

5. 매매/배당 기록 소소한 개선

- 매매기록/배당기록 필터 초기화 추가
- 배당 불러오기 0건일 때 안내 문구 개선

6. 안내 문서 보강

- 스크린샷 기반 유저가이드 추가: docs/USER_GUIDE_SCREENSHOT_V2_0.md
- v2.0에서 내부 구조를 어떻게 정리했는지 볼 수 있는 플로우차트 문서도 추가
- 그냥 파일만 던지는 게 아니라 처음 쓰는 사람도 대충 어디를 눌러야 하는지 따라갈 수 있게 정리함

주의

- 투자 판단 대신해주는 앱 아님
- Yahoo Finance / CNN JSON 기반이라 증권사 HTS/MTS랑 가격, 환율, 세금, 수수료 기준 차이 날 수 있음
- 코드 서명 없는 개인 제작 빌드라 윈도우 보안 경고 뜰 수 있음

이번 2.0은 기능 대폭 추가보다는 다음 기능들을 안정적으로 얹기 위한 정리 버전이라고 보면 됨.
