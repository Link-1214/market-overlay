# Market Overlay Release Archive

This directory keeps public distribution records for Market Overlay releases.

- Public repo contents are installer files, user-facing guides, patch notes, release posts, and checksums.
- GitHub automatically shows `Source code (zip/tar.gz)` on release pages, but this public repo is distribution-only and does not include the private Python source.
- Older releases are reconstructed from local archives where available. If an artifact was not preserved, the version README says `not archived`.

Current source development is maintained separately in the private repository.
