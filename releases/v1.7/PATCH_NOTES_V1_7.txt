Market Overlay v1.7 패치노트

v1.6 이후 변경 사항만 정리합니다.

1. 시장 상태 판정 개편
- 기존 과열/상승/평시/하락/폭락 표현을 극단적 탐욕/탐욕/중립/공포/극단적 공포로 변경했습니다.
- VIX 기준을 13 미만, 13~18, 18~24, 24~30, 30 이상으로 다시 나눠 중간 구간이 혼조로 빠지는 문제를 줄였습니다.
- 시장 상태 계산용 나스닥 축을 Nasdaq 100에서 Nasdaq Composite로 변경했습니다.
- F&G, IXIC/SPX 이동평균 괴리율, VIX를 가중 점수로 합산해 최종 시장 상태를 산출합니다.

2. CNN 공포탐욕지수 연동
- CNN dataviz JSON endpoint를 우선 사용하도록 변경했습니다.
- CNN 조회가 실패하면 기존 Local F&G 계산값으로 자동 대체합니다.
- 데이터 출처 표시에 CNN JSON 또는 Local F&G fallback 여부가 드러나도록 했습니다.
- CNN F&G 조회 성공 시 제공되는 과거 배열과 현재 스냅샷을 사용자 PC의 `%APPDATA%\MarketOverlay\fng_history.json`에 누적 저장하도록 했습니다.

3. 포트폴리오 화면 정리
- 상단 입력 영역과 카드 사이의 중간 보유종목 선택 목록을 숨겼습니다.
- 아래 보유종목 행을 더블클릭하면 해당 종목이 편집칸에 불러와집니다.
- 보유종목 표의 헤더를 클릭해 주요 칼럼 기준 오름차순/내림차순 정렬을 할 수 있게 했습니다.
- 포트폴리오 카드 아래에 비중 도넛 그래프와 색상 범례를 추가했습니다.
- 종합의 `종목별` 보기는 펼침 목록 대신 여러 포트의 같은 종목을 합산한 표로 바로 보여줍니다.
- 상세 표 칼럼을 `총보유금액`, `총수익금`, `수익률`, `비중` 순서로 정리했습니다.
- 비어 있는 `기본` 포트가 다른 포트 사용 중 반복 생성되어 보이는 일을 줄였습니다.

4. 매매기록 탭 정리
- 기존 `수익` 탭 이름을 `매매기록`으로 변경했습니다.
- 매도뿐 아니라 매수 거래도 자동 기록합니다. 매수 기록은 수량/단가 확인용이며 실현손익 합계에는 0원으로 반영됩니다.
- 기존 수익 기록은 매도 기록으로 호환해 읽습니다.
- 매매기록과 배당기록에 포트, 구분/통화/상태, 종목/메모 필터를 추가했습니다.
- 매매기록 상단의 종목 선택은 수동 추가 대상 선택과 종목 필터를 겸합니다. 특정 보유종목을 선택하면 해당 포트+종목 기록만 표시합니다.
- 매매기록과 배당기록의 칼럼 헤더를 클릭해 표시 행을 오름차순/내림차순으로 정렬할 수 있습니다.

5. 성과비교 탭 추가
- 현재 보유수량과 선택 기간의 가격 흐름을 기준으로 `종합` 또는 포트별 단순 성과를 비교하는 탭을 추가했습니다.
- 성과비교는 현재 보유 구성과 수량을 선택 기간 동안 보유했다고 가정해 계산합니다. 보유 시작일은 성과 기간을 줄이는 데 사용하지 않습니다.
- 비교 기간은 1개월, 3개월, 6개월, 1년, 3년 중 선택할 수 있습니다.
- 성과비교는 포트폴리오와 선택한 벤치마크의 수익률, 연환산 수익률, MDD를 표와 선그래프로 보여줍니다.
- 벤치마크는 IXIC, SPX, KOSPI, 금, 비트코인을 체크박스로 선택할 수 있습니다.
- IXIC, SPX, KOSPI 벤치마크 조회가 누락되던 심볼 인코딩 문제를 수정했습니다.
- 성과 그래프에 0% 기준선을 추가했습니다.
- MDD는 선택 기간 안에서의 고점 대비 최대 낙폭입니다.
- 선택 기간 데이터가 현저히 부족한 신규 상장/조회 실패 종목은 성과비교에서 제외하며, 제외 항목과 사유를 안내 문구에 표시합니다.
- 입출금, 매수 시점별 현금흐름, 세금은 아직 제외한 1차 비교 기능입니다.

6. 종목명 데이터 갱신
- 미국 상장 심볼, 국내 상장사, KRX ETF/ETN 표시명 데이터를 최신 조회 기준으로 다시 생성했습니다.
- EDHL, RTX, PL, IMSR 및 국내 ETF/ETN 주요 종목의 표시명 매핑을 확인했습니다.

7. 시작 표시 버그 수정
- 설치 확인 과정에서 숨김 상태로 실행되면 오버레이만 보이고 컨트롤 창이 보이지 않을 수 있던 문제를 보강했습니다.
- 앱 시작 직후 컨트롤 창을 여러 차례 명시적으로 표시하고 활성화하도록 했습니다.

8. 설치/제거 흐름 개선
- 설치파일 실행 시 기존 설치가 감지되면 `설치/업데이트`, `제거`, `종료` 버튼으로 작업을 선택합니다.
- 제거를 선택하면 개인 데이터 삭제 여부를 묻는 기존 언인스톨 흐름으로 연결됩니다.
- 개인 데이터에는 포트폴리오, 보유종목, 예수금, 매매/배당 기록, 사용자 표시명, 설정값이 포함됩니다.

9. 다음 버전 후보 정리
- 맥 버전은 v1.8 이후 별도 실험 빌드로 분리합니다.
- 입출금 기반 XIRR, 장기 백테스트, 기간별 히트맵은 후속 분석 기능 후보로 보류했습니다.
