★ Market Overlay v1.7 또 개선해옴

다운로드 링크
(구글드라이브 링크 넣기)

AI Agent / Codex랑 계속 만지면서 만든 데스크톱 투자 보조툴임.
초단위 매매 프로그램은 아니고, 시장 상태랑 내 포트폴리오 흐름을 화면에서 가볍게 확인하려고 만든 쪽에 가까움.

이번 v1.7은 1.6에서 꽤 많이 손봤음.
이제 슬슬 개인용 장난감에서 벗어나서 실제로 계속 켜놓고 쓰기 편한 쪽으로 다듬는 중.


주요 변경점

1. CNN 공포탐욕지수 연동

- 기존 Local F&G 대신 CNN dataviz 쪽 데이터를 우선으로 불러오게 바꿈
- CNN 쪽 조회가 실패하면 Local F&G로 대체
- 오버레이 표기도 Local F&G가 아니라 CNN F&G 기준으로 보이도록 수정
- 공탐 데이터는 사용자 PC에 스냅샷처럼 누적 저장됨

※ 그래도 CNN 쪽이 막히거나 응답이 이상하면 일시적으로 Local 계산값이 나올 수 있음


2. 시장 상태 표현 변경

- 기존 과열/상승/평시/하락/폭락 표현을
  극단적 탐욕 / 탐욕 / 중립 / 공포 / 극단적 공포 로 변경
- 나스닥 기준도 Nasdaq 100이 아니라 Nasdaq Composite 쪽으로 수정
- VIX 구간도 다시 나눠서 혼조가 너무 자주 뜨는 문제를 줄임


3. 포트폴리오 UI 정리

- 상단에 포트 추가/이름 변경/삭제/조회 버튼이 많아서 복잡했던 부분을 줄임
- 포트폴리오 탭 오른쪽 + 버튼으로 포트 추가
- 포트 이름은 탭 더블클릭으로 수정
- 포트 삭제는 탭에 마우스 올리면 X가 뜨고, 삭제 전 확인창이 나옴
- 종목 편집은 아래 보유 목록에서 더블클릭해서 불러오는 방식으로 정리

화면이 기능 늘어난 만큼 복잡해지는 게 싫어서, 최대한 탭 중심으로 줄였음.


4. 포트폴리오 보기 개선

- 종합 화면에서 포트별 / 종목별 보기 가능
- 포트별은 비중 큰 순서로 정렬
- 종목별은 같은 종목이 여러 포트에 있어도 통합해서 볼 수 있음
- 도넛 그래프로 포트 비중을 볼 수 있게 추가
- 보유 종목 표에 총수익금, 수익률, 비중 표시 추가


5. 매매기록 / 배당기록 개선

- 기존 수익 탭을 매매기록 탭으로 바꿈
- 매수/매도 기록이 자동으로 남고, 필요하면 수동 추가도 가능
- 매매기록과 배당기록에 필터 추가
- 날짜, 종목, 금액 같은 칼럼을 눌러 정렬 가능
- 배당은 중복 기록을 줄이도록 최근 기록 기준으로 불러오는 방식 유지


6. 성과비교 탭 추가

- 현재 보유 종목과 수량을 선택한 기간 동안 들고 있었다고 가정해서 단순 성과 비교
- 기간은 1개월 / 3개월 / 6개월 / 1년 / 3년
- 비교 대상은 IXIC, S&P500, KOSPI, 금, 비트코인 체크박스로 선택 가능
- 수익률, 연환산, MDD 표시
- 그래프에 0% 기준선 추가
- 데이터가 부족해서 제외된 종목은 어떤 종목이 빠졌는지 안내

※ 입출금, 매수 시점별 현금흐름, 세금, 수수료까지 반영한 정밀 성과 분석은 아직 아님


7. 종목명 데이터 갱신

- 미국 상장 심볼 12,675개
- 국내 상장사 2,717개
- KRX ETF 1,130개
- ETN 383개
- 총 16,905개 매핑으로 갱신함

ETF류 이름이 티커로만 뜨거나 이상하게 뜨던 문제를 꽤 줄였음.
그래도 너무 신규거나 마이너한 종목은 이상하게 나올 수 있음.


8. 설치 / 제거 개선

- 설치파일을 다시 실행하면 기존 설치를 감지해서 업데이트 / 제거 / 종료를 선택할 수 있게 함
- 제거할 때 사용자 개인 데이터를 삭제할지 물어봄
- 사용자 데이터에는 포트폴리오, 예수금, 표시명 보정, 매매기록, 배당기록, 설정값 등이 포함됨
- 기본 저장 위치는 %APPDATA%\MarketOverlay


주의사항

- 이 프로그램은 투자 판단을 대신해주는 도구가 아님
- Yahoo Finance, CNN 등 외부 데이터를 참고하므로 지연, 누락, 일시 실패가 있을 수 있음
- 증권사 HTS/MTS와 수수료, 세금, 환율, 배당 처리 방식이 다를 수 있음
- 설치파일은 코드 서명이 없어서 SmartScreen, V3 같은 보안 프로그램에서 경고가 뜰 수 있음
- 중요한 매매 판단은 반드시 증권사 화면과 대조해서 확인하는 걸 추천함


설치 방법

1. 압축을 풀고 MarketOverlaySetup_v1.7.exe 실행
2. 기존 버전이 있으면 업데이트 선택
3. 설치 후 Market Overlay 실행
4. 포트폴리오 탭에서 내 보유 종목과 예수금 입력
5. 오버레이는 시장 탭에서 잠금/숨김/새로고침 조정

업데이트 설치 시 기존 개인 데이터는 기본적으로 보존됨.
완전 삭제하고 싶으면 제거 과정에서 사용자 데이터 삭제를 선택하면 됨.
