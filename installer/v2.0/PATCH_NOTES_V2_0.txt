Market Overlay v2.0 패치노트

이번 버전은 v1.9 이후 변경된 내용만 정리했습니다.

1. 코드 구조 정리

- 단일 실행 파일 구조는 유지하면서 계산, 저장, 포맷, 종목명, 히스토리 캐시, worker payload, UI 조립 책임을 분리했습니다.
- `models.py`, `formatters.py`, `symbol_names.py`, `state_store.py`, `portfolio_calculations.py`, `history_cache.py`, `worker_payloads.py`, `ui_helpers.py`, `control_tab_builders.py`, `portfolio_tab_builders.py`를 추가했습니다.
- 구조 정리 후에도 포트폴리오 탭, 오버레이, 매매/배당 기록, 성과비교, 히트맵의 기존 동작을 유지하도록 회귀 테스트를 보강했습니다.

2. 설치/제거 UX 보강

- 기존 설치 감지를 HKCU/HKLM uninstall registry와 실제 `unins000.exe` fallback 기준으로 보강했습니다.
- 설치 중 기존 버전이 있으면 설치/업데이트, 제거, 종료 선택지를 더 명확하게 안내합니다.
- 제거 시 개인 데이터에 포함되는 항목을 안내하고 보존/삭제 판단을 돕도록 문구를 정리했습니다.
- 설치 등록 정보에 GitHub 지원/업데이트 URL과 제거 아이콘 정보를 보강했습니다.

3. 백업/복원 안내 개선

- 백업 내보내기/가져오기 문구를 사용자 데이터 범위와 자동 백업 흐름 중심으로 다듬었습니다.
- 가져오기 실패 시 기존 데이터 복구 흐름을 코드와 테스트로 확인했습니다.
- 실제 사용자 데이터 해시를 보존한 상태에서 샘플 데이터 교체/복원 검증을 진행했습니다.

4. 종목명 데이터 갱신

- 내장 종목명 데이터를 갱신했습니다.
- `SPCX`는 별도 큐레이션으로 `스페이스X` 표시명을 추가했습니다.
- 사용자 표시명 보정은 계속 내장 데이터보다 우선합니다.

5. 매매/배당 기록 UX 보강

- 매매기록과 배당기록 필터 초기화 버튼을 추가했습니다.
- 배당 불러오기 결과가 0건일 때 범위 내 배당 없음/중복 제외 가능성을 더 분명히 표시합니다.
- 드롭다운 wheel 오작동 방지, 필터/정렬 회귀 테스트를 유지합니다.

6. 문서와 GitHub 운영 정리

- public repo는 설치파일과 사용자 문서만 담고, private repo는 소스와 개발 문서를 관리하는 구조로 정리했습니다.
- 스크린샷 기반 사용자 가이드를 보강했습니다.
- v1.1~v1.9 과거 배포 자료는 가능한 범위에서 public/private 양쪽에 아카이브했습니다.

검증

- `tools\pre_release_check.ps1` 전체 게이트 통과 기준으로 배포합니다.
- 설치 전후 `%APPDATA%\MarketOverlay\market_overlay_state.json` 해시 보존을 확인합니다.
- 설치 후 실행, 중복 실행 방지, 바로가기/설치 등록 정보를 확인합니다.
