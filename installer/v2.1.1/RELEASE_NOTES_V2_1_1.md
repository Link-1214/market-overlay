# Market Overlay v2.1.1

v2.1.1은 미국·국내·일본 상장 종목 자동완성 목록을 공식 출처 기준으로 다시 갱신한 종목 데이터 패치입니다.

## 주요 변경

- 미국 상장 종목 12,807개 재검증
- 코스피·코스닥·코넥스 상장법인 2,711개 재검증
- KRX ETF 1,140개, ETN 380개 반영
- JPX 공식 목록 4,445개 반영
- `6222.T`, `7811.T`, `130A.T` 등 기존 누락 일본 종목 추가
- 통합 종목 사전 17,113개 → 21,484개

## 갱신 안전장치

- 시장별 최소 건수와 대표 종목을 자동 검증합니다.
- 소스 다운로드나 파싱에 실패하면 기존 데이터를 덮어쓰지 않습니다.
- 이 검증은 향후 기능 업데이트와 종목 데이터 전용 패치에서 매번 실행합니다.

## 안내

- 기존 v2.1 설정과 개인 장부는 업데이트 설치 후에도 유지됩니다.
- 사용자가 직접 보정한 종목 표시명은 내장 데이터보다 우선합니다.
