Market Overlay v2.1 올림

이번 버전은 일본 종목/JPY 지원이랑 업데이트 확인을 넣고,
안쪽 코드를 한 번 더 크게 정리한 버전임.

주요 변경

1. 일본 종목 지원
- 7203 같은 일본 4자리 코드와 `.T` 심볼 지원
- JPY 평단가/예수금과 원화 환산 반영
- 자동완성은 한국, 미국, 일본 순으로 정렬

2. 트리맵 정리
- 이름부터 히트맵이 아니라 트리맵으로 바로잡음
- 미국은 티커, 한국/일본은 종목명으로 표시
- 작은 박스 글자 겹침 줄이고 마우스 올리면 상세정보 나오게 함

3. 데이터 보정
- Yahoo 일봉에서 전일 종가가 잠깐 비는 경우에만 60분봉으로 보정
- CNN 공탐은 기존처럼 공식 JSON 우선, 실패 시 Local F&G 사용

4. 업데이트 확인
- 하루 한 번 GitHub 정식 릴리즈만 확인
- 새 버전이 있을 때만 알림
- 릴리즈 페이지 열기 / 이번 버전 알림 끄기 / 나중에 선택 가능
- 자동 설치는 안 하고 사용자가 직접 결정하게 둠

5. 내부 구조 정리
- 시각화, 오버레이 3종, 데이터 worker, 중복실행/종료 처리를 각각 파일로 분리
- 앱은 여전히 실행파일 하나임
- 겉으로 확 달라지는 작업은 아니지만 다음 수정 때 덜 꼬이게 만드는 게 핵심

설치

- GitHub Release의 `MarketOverlaySetup_v2.1.exe` 실행
- 기존 버전 위에 설치해도 `%APPDATA%\MarketOverlay`의 장부와 설정은 유지됨

주의

- 투자판단 대신해주는 앱 아님
- Yahoo Finance/CNN JSON 기반이라 증권사 값과 시차나 기준 차이가 날 수 있음
- 개인 제작 무서명 빌드라 Windows 보안 경고가 뜰 수 있음
