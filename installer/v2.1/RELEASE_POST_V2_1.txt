Market Overlay v2.1 올림

이번 버전은 실제 사용 중 발견된 전일대비 오류 가능성을 먼저 보정하고,
트리맵 가독성, 일본 종목/JPY, 업데이트 확인까지 보강한 버전임.

주요 변경

1. 데이터 보정
- 실제로 원본 데이터 소스 자체가 잠시 비어 전일대비가 정상적으로 출력되지 않는 사례가 있었음
- 같은 문제가 생기지 않도록 Yahoo 일봉에서 전일 종가가 잠시 비면 60분봉으로 보정하도록 함
- 정상적으로 전일 종가가 들어오는 경우에는 추가 조회하지 않음

2. 트리맵 가독성 개선
- 이름부터 히트맵이 아니라 트리맵으로 바로잡음
- 작은 박스에서 글자가 겹치지 않도록 표시 항목과 배치를 조정함
- 미국 종목은 티커, 한국/일본 종목은 종목명 위주로 보여줘 한눈에 알아보기 쉽게 함
- 박스에 다 못 넣는 정보는 마우스를 올리면 종목명, 코드, 수익률, 비중, 평가액으로 확인 가능

3. 일본 종목 지원
- 7203 같은 일본 4자리 코드와 `.T` 심볼 지원
- JPY 평단가/예수금과 원화 환산 반영
- 자동완성은 한국, 미국, 일본 순으로 정렬

4. 업데이트 확인
- 하루 한 번 GitHub 정식 릴리즈만 확인
- 새 버전이 있을 때만 알림
- 릴리즈 페이지 열기 / 이번 버전 알림 끄기 / 나중에 선택 가능
- 자동 설치는 안 하고 사용자가 직접 결정하게 둠

5. 백업/복원 안내 개선
- 어떤 개인 데이터가 백업되는지, 가져오면 현재 데이터가 어떻게 바뀌는지 안내를 보강함
- 가져오기 전 기존 상태를 자동 백업하는 흐름은 그대로 유지함

6. 내부 구조 정리
- 시각화, 오버레이 3종, 데이터 worker, 중복실행/종료 처리를 각각 파일로 분리하고 내부 실행 코드를 보강함
- 앱은 여전히 실행파일 하나임
- 겉으로 확 달라지는 작업은 아니지만 다음 수정 때 덜 꼬이게 만드는 게 핵심

설치

- GitHub Release의 `MarketOverlaySetup_v2.1.exe` 실행
- 기존 버전 위에 설치해도 `%APPDATA%\MarketOverlay`의 장부와 설정은 유지됨

주의

- 투자판단 대신해주는 앱 아님
- Yahoo Finance/CNN JSON 기반이라 증권사 값과 시차나 기준 차이가 날 수 있음
- 개인 제작 무서명 빌드라 Windows 보안 경고가 뜰 수 있음
