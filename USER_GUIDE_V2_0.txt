Market Overlay v2.0 사용자 안내

Market Overlay는 시장 상태와 내 포트폴리오 흐름을 가볍게 확인하는 데스크톱 보조 도구입니다.
투자 판단을 대신하지 않으며, Yahoo Finance/CNN JSON 기반 조회값은 증권사 HTS/MTS와 차이가 날 수 있습니다.

1. 실행과 오버레이

- 앱을 실행하면 컨트롤 창과 오버레이가 함께 열립니다.
- 시장 탭에서 `오버레이 잠금`, `전체 새로고침`, `오버레이 숨기기`를 사용할 수 있습니다.
- 잠금 상태에서는 오버레이가 항상 위에 떠 있으면서 뒤쪽 창 클릭을 방해하지 않습니다.
- 설정 탭의 `오버레이 화면`에서 `시장 오버레이`, `종합 오버레이`, `내 포트폴리오 오버레이`를 선택할 수 있습니다.

2. 시장 탭

- CNN F&G, VIX, USD/KRW, NASDAQ Composite, S&P 500, KOSPI를 확인합니다.
- CNN F&G 자체 구분은 `0~24 극단적 공포`, `25~44 공포`, `45~55 중립`, `56~75 탐욕`, `76~100 극단적 탐욕`입니다.
- 앱의 `시장 상태`는 F&G, IXIC/SPX 이동평균, VIX를 함께 보는 별도 요약입니다.
- 시장 데이터는 자동 갱신되며, 수동으로는 `전체 새로고침`을 누르면 됩니다.

3. 포트폴리오 탭

- 상단 포트폴리오 탭에서 포트를 선택합니다.
- `+` 탭으로 포트를 추가하고, 포트 탭 이름을 더블클릭해 이름을 바꿀 수 있습니다.
- 포트 탭에 마우스를 올리면 삭제 버튼이 표시되며, 삭제 전 확인 창이 뜹니다.
- 종목 입력칸은 티커나 국내 6자리 코드를 입력하면 가능한 표시명을 자동완성합니다.
- 추가/수정, 매수/매도, 예수금 저장을 통해 보유 수량과 현금 상태를 관리합니다.
- 예수금은 음수가 될 수 있으며, 음수일 때 현금비중 카드가 경고색으로 표시됩니다.
- 종합 보기에서는 포트별/종목별 롤업을 비중 내림차순으로 확인할 수 있습니다.
- 좌측 시각화 영역에서 비중 도넛과 기간별 히트맵을 전환할 수 있습니다.

4. 성과비교 탭

- 현재 보유 종목과 현재 보유수량을 선택 기간 동안 그대로 보유했다고 가정한 단순 성과 비교입니다.
- 기간은 1개월, 3개월, 6개월, 1년, YTD 중 선택합니다.
- 비교 기준은 IXIC, S&P 500, KOSPI, 금, 비트코인을 체크박스로 켜고 끌 수 있습니다.
- MDD는 선택 기간 안에서 고점 대비 가장 크게 빠진 구간의 낙폭입니다.
- 조회 불가 또는 기간 데이터가 부족한 종목은 제외 목록에 표시됩니다.

5. 매매기록 탭

- 매수/매도 기록은 포트폴리오 탭의 거래 입력과 연동됩니다.
- 수동 추가도 가능하며, 포트/구분/통화/상태는 드롭다운으로 관리합니다.
- 매수단가, 매도단가, 수량, 날짜는 사용자가 입력하고, 손익/수익률은 자동 계산됩니다.
- 필터와 칼럼 정렬을 사용해 원하는 기록만 확인할 수 있습니다.

6. 배당 탭

- 보유 종목의 배당 이력을 불러오거나 수동 추가할 수 있습니다.
- 중복 기록은 날짜/포트/종목/통화/배당 단가 기준으로 걸러집니다.
- 배당 불러오기 결과가 0건이면 중복 제외인지, 범위 내 배당이 없는지 안내 문구가 표시됩니다.

7. 설정 탭

- 이동평균 기준, 스파크라인 표시, 조회 대상 지수, 자동 갱신 주기, 오버레이 화면을 조정합니다.
- 백업 내보내기는 현재 개인 데이터를 별도 파일로 저장합니다.
- 백업 가져오기는 적용 전 자동 백업을 만들고, 실패 시 기존 데이터를 복구합니다.
- 개인 데이터에는 포트폴리오, 보유종목, 예수금, 매매/배당 기록, 사용자 표시명, 앱 설정이 포함됩니다.

8. 저장 위치와 업데이트

- 개인 데이터는 `%APPDATA%\MarketOverlay`에 저장됩니다.
- 프로그램은 `%LOCALAPPDATA%\Programs\Market Overlay`에 설치됩니다.
- 업데이트 설치는 개인 데이터를 보존합니다.
- 제거 시 개인 데이터 삭제 여부를 묻는 안내가 표시됩니다.

주의

- 이 앱은 투자 판단을 대신하지 않는 보조 도구입니다.
- 데이터 출처 지연, 환율 기준, 세금/수수료 반영 여부에 따라 실제 증권사 값과 차이가 날 수 있습니다.
- 코드 서명이 없는 빌드라 Windows SmartScreen 또는 보안 프로그램 경고가 표시될 수 있습니다.
