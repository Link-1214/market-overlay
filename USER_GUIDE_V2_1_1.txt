Market Overlay v2.1.1 사용자 안내

v2.1.1은 종목 데이터 최신화 패치입니다. 기존 v2.1 기능과 사용법은 동일하며, 미국·코스피·코스닥·도쿄증권거래소 종목 자동완성 목록을 공식 출처 기준으로 갱신했습니다.

Market Overlay는 시장 상태와 개인 포트폴리오 흐름을 확인하는 Windows 데스크톱 보조 도구입니다.

1. 실행과 오버레이

- 앱을 실행하면 컨트롤 창과 선택한 오버레이가 함께 열립니다.
- 설정 탭에서 시장, 종합, 내자산 오버레이를 선택합니다.
- 잠금 상태에서는 오버레이가 항상 위에 떠 있으면서 뒤쪽 창 클릭을 통과시킵니다.
- 같은 앱을 다시 실행하면 새 프로세스를 늘리지 않고 기존 컨트롤 창을 앞으로 가져옵니다.

2. 시장 탭

- CNN F&G, VIX, USD/KRW, Nasdaq Composite, S&P 500, KOSPI와 선택 지표를 확인합니다.
- CNN F&G 구분은 0~24 극단적 공포, 25~44 공포, 45~55 중립, 56~75 탐욕, 76~100 극단적 탐욕입니다.
- 시장 상태는 F&G, IXIC/SPX 이동평균 괴리, VIX를 함께 계산한 별도 요약입니다.
- Yahoo 일봉 전일가가 일시적으로 비면 앱이 같은 chart API의 60분봉으로 조건부 보정합니다.

3. 포트폴리오와 일본 종목

- 종목 입력칸에 티커, 국내 6자리 코드, 일본 4자리 코드 또는 종목명을 입력해 자동완성을 사용합니다.
- 일본 종목은 Yahoo `.T` 심볼로 저장되며 기본 거래 통화는 JPY입니다. 예: `7203` 또는 `7203.T`.
- 평단가/예수금 통화에서 JPY를 선택할 수 있고 총액은 JPY/KRW 환율로 원화 환산됩니다.
- `+` 탭으로 포트를 추가하고 포트 탭 더블클릭으로 이름을 수정합니다.
- 매수/매도, 직접 수정, 삭제와 통화별 예수금 저장을 지원합니다.

4. 비중과 트리맵

- 포트폴리오 탭 좌측에서 `비중`과 `트리맵`을 전환합니다.
- 트리맵 박스 크기는 현재 평가액 비중, 색은 선택 기간 수익률입니다.
- 미국 종목은 티커, 국내/일본 종목은 종목명을 박스 안에 표시합니다.
- 마우스를 올리면 종목명, 코드, 기간 수익률, 비중, 평가액을 확인할 수 있습니다.
- 작은 박스는 수익률 중심으로 표시하며 들어가지 않는 정보는 tooltip에서 확인합니다.

5. 성과비교

- 현재 보유 종목과 수량을 선택 기간 내내 보유했다고 가정한 단순 비교입니다.
- 1개월, 3개월, 6개월, 1년, YTD와 IXIC/S&P 500/KOSPI/금/비트코인을 선택할 수 있습니다.
- MDD는 선택 기간 내 고점 대비 최대 낙폭입니다.
- 반영/제외 항목 수와 데이터 부족 또는 조회 실패 사유를 화면에서 확인합니다.

6. 매매기록과 배당

- 매매기록은 포트폴리오 거래와 연동되며 필터, 정렬, 수동 입력을 지원합니다.
- 배당은 보유 이력 기준으로 불러오며 중복 기록을 제외합니다.
- JPY 기록은 별도 카드를 추가하지 않고 총 실현손익/총 배당에 원화 환산해 반영합니다.

7. 백업과 복원

- 설정 탭의 백업 내보내기는 포트, 보유종목, 예수금, 매매/배당 기록, 사용자 표시명과 앱 설정을 저장합니다.
- 가져오기는 현재 데이터를 병합하지 않고 교체하며, 적용 전에 기존 상태를 자동 백업합니다.
- 설치 프로그램과 시장 가격 캐시는 백업 대상이 아닙니다.
- 개인 데이터 위치: `%APPDATA%\MarketOverlay`

8. 업데이트 확인

- 자동 확인은 기본 켜짐이며 실행 후 하루 한 번 GitHub 정식 Release를 확인합니다.
- 업데이트가 없거나 자동 확인이 실패하면 팝업이 뜨지 않습니다.
- 새 버전 알림에서 릴리즈 페이지 열기, 이번 버전 알림 끄기, 나중에를 선택합니다.
- 특정 버전을 숨겨도 더 높은 버전이 나오면 다시 알립니다.
- 설정 탭의 `업데이트 확인`은 24시간 제한과 숨김 설정을 무시하고 즉시 확인합니다.
- 앱은 자동 다운로드/자동 설치를 하지 않습니다.

9. 설치와 제거

- 설치 위치: `%LOCALAPPDATA%\Programs\Market Overlay`
- 업데이트 설치는 `%APPDATA%\MarketOverlay`의 개인 데이터를 보존합니다.
- 제거 시 개인 데이터도 삭제할지 별도로 선택할 수 있습니다.

주의

- 투자 판단을 대신하는 도구가 아닙니다.
- 데이터 지연과 환율 기준 때문에 증권사 값과 차이가 날 수 있습니다.
- 코드 서명이 없어 Windows 보안 경고가 표시될 수 있습니다.
