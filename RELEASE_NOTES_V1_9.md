# Market Overlay v1.9.0

v1.9는 오버레이 모드와 실행 안정성을 크게 보강한 공개 첫 릴리즈입니다.

## 다운로드

- `MarketOverlaySetup_v1.9.exe`를 다운로드해 실행하세요.
- 설치 후 `Market Overlay`를 실행하면 됩니다.
- 업데이트 설치는 기존 개인 포트폴리오 데이터를 보존합니다.

## 주요 변경

- 오버레이 화면 모드 3종 추가/정리
  - 시장 오버레이
  - 종합 오버레이
  - 내 포트폴리오 오버레이
- 종합 오버레이 추가
  - 시장요약, 공탐/VIX, S&P/NASDAQ, KOSPI/USD-KRW, 포트폴리오 핵심 수치를 압축 표시
- 내 포트폴리오 오버레이 추가
  - 총평가액, 원금, 수익, 전일대비, 포트 비중, 투자/현금 비중 요약
- 중복 실행 방지
  - 이미 실행 중이면 새 프로세스를 늘리지 않고 기존 앱을 활성화
- CNN Fear & Greed 구간 기준 보정
- 종합 오버레이 금액 표기 만원 단위 압축
