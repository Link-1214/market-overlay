Market Overlay v2.2 패치노트
============================

v2.2는 신뢰성ㆍ진단 릴리즈입니다. 큰 계산 정책은 바꾸지 않고, 데이터 조회 경계와 상태 저장, 로그, 시각화 회귀 검증을 보강했습니다.

1. 시장 데이터 조회 안정화
- Yahoo 가격/배당/히스토리, CNN Fear & Greed, USD/KRWㆍJPY/KRW 환율 조회를 market data provider로 분리했습니다.
- Yahoo 일봉의 전일 종가가 잠시 비는 경우 5일 60분봉 데이터로 보정하는 기존 정책을 provider 경계 안으로 정리했습니다.
- 최초 조회 실패 시 숫자를 무리하게 표시하지 않고, 이전 정상값이 있으면 마지막 정상값을 유지하는 정책을 보존했습니다.

2. 상태 저장 schema 도입
- 사용자 state에 schema_version을 추가했습니다.
- 구버전 state는 자동 이관되며, heatmap 관련 설정은 treemap 설정으로 호환 이동합니다.
- 업데이트 확인, JPY, 포트폴리오 시각화 splitter 설정이 없으면 기본값을 채웁니다.
- 백업 export/import에도 schema 정보가 포함됩니다.

3. 진단 로그 추가
- `%APPDATA%\MarketOverlay\logs`에 순환 로그를 저장합니다.
- 네트워크, 파싱, state, worker, 오버레이 복구 문제를 추적할 수 있게 했습니다.
- 보유수량, 평단가, 예수금, 장부 세부 금액은 로그에 남기지 않습니다.
- 설정 탭에 `로그 폴더 열기` 버튼을 추가했습니다.

4. 포트폴리오 시각화 영역 조절
- 포트폴리오 탭의 카드 아래 영역을 좌우 분할로 바꿨습니다.
- 왼쪽은 `비중/트리맵`, 오른쪽은 보유 목록/종합 롤업 목록입니다.
- 분할선은 드래그로 조절할 수 있고, 다음 실행 때 복원됩니다.
- `기본값` 버튼으로 기본 비율로 되돌릴 수 있습니다.

5. 시각 회귀 스모크 테스트
- 시장 오버레이, 종합 오버레이, 내자산 오버레이, 비중 보기, 트리맵, 성과비교 화면의 offscreen 렌더 검증을 추가했습니다.
- 빈 화면, 핵심 문구 누락, 트리맵 박스 미렌더링 같은 회귀를 배포 전 자동으로 잡도록 했습니다.

6. 종목명 데이터 최신화
- NASDAQ/NYSE/AMEX, KRX 상장법인/ETF/ETN, JPX 전체 목록을 다시 수집했습니다.
- 검증 결과: 미국 12,881개, KRX 4,231개, JPX 4,446개, 총 21,558개 매핑.

7. 내부 코드 구조 개선
- 시장 데이터 provider, state schema, 진단 로그, 시각화 스모크 테스트, 포트폴리오 splitter 처리를 책임별로 분리했습니다.
- 앱은 여전히 하나의 Windows 프로그램으로 설치ㆍ실행됩니다.

주의
- 이 앱은 투자 판단을 대신하지 않는 보조 도구입니다.
- Yahoo Finance 기반 조회값은 증권사 HTS/MTS와 지연, 환율, 세금, 수수료 기준이 다를 수 있습니다.
- 코드 서명이 없는 개인 빌드라 Windows SmartScreen 또는 보안 프로그램 경고가 표시될 수 있습니다.
