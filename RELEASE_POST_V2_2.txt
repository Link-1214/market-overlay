Market Overlay v2.2 올림
========================

이번 버전은 기능을 크게 늘린 버전이라기보다는 신뢰성/진단 릴리즈에 가까움.

가장 큰 변경은 데이터 조회와 내부 구조 정리임.

1. 데이터 보정/조회 안정화
- 실제로 Yahoo 원본 일봉에서 전일 종가가 잠시 비는 케이스가 있었고, 그때 전일대비가 이상하게 나오는 문제가 있었음.
- 이번 버전에서는 Yahoo 일봉 전일 종가가 비면 5일 60분봉 데이터로 보정하도록 조회 구조를 정리함.
- 시장 데이터 조회 부분을 별도 provider로 빼서 이후 오류 추적이나 수정이 쉬워지게 함.

2. 트리맵/비중 화면 개선
- 포트폴리오 탭의 비중/트리맵 영역을 드래그로 넓히거나 줄일 수 있게 함.
- 왼쪽은 시각화, 오른쪽은 보유 목록으로 나뉘고, 기본값 버튼으로 원래 비율로 되돌릴 수 있음.
- 트리맵은 박스 크기 = 현재 자산 비중, 색상 = 선택 기간 수익률.

3. 진단 로그 추가
- 문제가 생겼을 때 원인을 좀 더 찾기 쉽게 로그를 남기도록 함.
- 단, 보유수량/평단가/예수금 같은 장부 세부값은 로그에 남기지 않도록 처리함.
- 설정 탭에서 로그 폴더를 열 수 있음.

4. 종목명 데이터 최신화
- 미국 상장, KRX 상장법인/ETF/ETN, 일본 JPX 종목 데이터를 다시 수집함.
- 총 21,558개 종목명 매핑.

5. 내부 구조 정리
- 시각화, 시장 데이터 조회, state 이관, 진단 로그 쪽을 더 분리함.
- 앱은 여전히 하나의 Windows 설치 프로그램으로 설치됨.

주의
- 투자 판단을 대신하는 도구는 아님.
- Yahoo Finance 기반이라 증권사 HTS/MTS와 지연, 환율, 세금, 수수료 기준 차이가 있을 수 있음.
- 코드서명 없는 개인 빌드라 SmartScreen이나 보안 프로그램 경고가 뜰 수 있음.

기존 사용자 데이터는 업데이트 설치해도 유지됨.
