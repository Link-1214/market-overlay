# Market Overlay v2.2

v2.2는 포트폴리오 시각화 조작감을 보강한 신뢰성ㆍ진단 릴리즈입니다. 사용자가 바로 체감하는 변화는 포트폴리오 탭의 `비중/트리맵` 영역을 직접 넓히고 줄일 수 있게 된 점이며, 내부적으로는 시장 데이터 조회, state 이관, 진단 로그, 시각화 회귀 검증을 보강했습니다.

## 주요 변경

- 포트폴리오 탭의 `비중/트리맵` 시각화와 보유 목록을 좌우 splitter로 조절할 수 있게 했습니다.
- 시각화 영역은 `기본값` 버튼으로 되돌릴 수 있고, 조절한 폭은 다음 실행 때 복원됩니다.
- v2.1에서 추가한 Yahoo 전일 종가 결측 보정은 유지하고, v2.2에서는 Yahoo/CNN/환율 조회를 market data provider로 분리해 조회 경계를 정리했습니다.
- 기존 전일가 보정 경로와 CNN fallback, 환율 조회를 회귀 테스트와 진단 로그 대상에 포함했습니다.
- state `schema_version`과 legacy migration을 도입했습니다.
- `%APPDATA%\MarketOverlay\logs` 진단 로그와 설정 탭의 `로그 폴더 열기`를 추가했습니다.
- 시장/종합/내자산 오버레이, 비중, 트리맵, 성과비교 offscreen 스모크 테스트를 추가했습니다.
- 내장 종목명 데이터를 최신화했습니다. 미국 12,881개, KRX 4,231개, JPX 4,446개, 총 21,558개입니다.

## 주의

- 사용자 데이터는 업데이트 설치 중 `%APPDATA%\MarketOverlay`에 유지됩니다.
- 앱은 업데이트를 자동 다운로드하거나 자동 설치하지 않습니다.
- 코드 서명이 없는 개인 빌드라 Windows SmartScreen 또는 보안 프로그램 경고가 표시될 수 있습니다.
